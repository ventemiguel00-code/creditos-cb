2. Estructura por pestañas

Todo el sistema esta organizado en pestañas para que el acceso sea mas comodo desde celular y computador.

Resumen

Muestra el capital prestado, ganancia neta, total recaudado y capital inicial disponible.

Clientes

Registro, listado, busqueda, foto, fecha de creacion y eliminacion de clientes.

Prestamos

Creacion, edicion, eliminacion, porcentaje configurable y estado del prestamo.

Pagos

Registro de pagos, recibo imprimible y actualizacion del saldo restante.

Configuracion

Cambio de clave, limpieza del historial y descarga del archivo Excel.

3. Registro de clientes

Desde la pestaña Clientes puedes crear nuevos clientes con sus datos completos.

  1. Ingresa el nombre del cliente.
  2. Escribe direccion, telefono y correo si aplica.
  3. Adjunta una foto desde camara o archivo.
  4. Presiona el boton para guardar.
Dato importante El sistema muestra la fecha con texto mas claro, por ejemplo: Creado 19/04/2026.

4. Creacion de prestamos

Al crear un prestamo, el sistema calcula automaticamente el total a pagar y el valor de cada cuota usando el porcentaje que definas.

  1. Selecciona el cliente.
  2. Ingresa el monto prestado.
  3. Ingresa el numero de cuotas.
  4. Define el porcentaje de interes.
  5. Guarda el prestamo.

Total a pagar

Lo calcula Supabase segun las columnas generadas de tu tabla.

Valor cuota

Se toma automaticamente desde la base de datos.

Estado

Puede mostrarse como Al dia o Pago de prestamo completado.

5. Edicion y eliminacion

La app permite administrar tanto clientes como prestamos cuando sea necesario.

  • Puedes editar un prestamo para corregir monto, cuotas o porcentaje.
  • Puedes borrar un prestamo desde su seccion correspondiente.
  • Puedes borrar un cliente y el sistema elimina tambien la informacion relacionada para evitar inconsistencias.
Recomendacion Antes de borrar informacion importante, descarga primero el Excel como respaldo.

6. Registro de pagos y recibos

Cada pago puede generar un recibo listo para imprimir desde la misma app, sin depender de ventanas emergentes del navegador.

  • Se muestra la foto del cliente.
  • Se registra la fecha del pago.
  • Se indica el valor pagado.
  • Se muestra la cuota actual y el total de cuotas.
  • Se informa el saldo restante.

7. Capital inicial editable

En el tablero principal existe un cuadro para definir el capital inicial de trabajo. Ese valor se va comparando con los prestamos registrados.

  • Si la tabla configuracion_app esta activa, se guarda para todos los dispositivos.
  • Si no esta disponible, se guarda localmente en el navegador.
  • La app indica si el valor esta guardado de forma compartida o solo local.

8. Limpieza automatica del historial

Para ahorrar almacenamiento, la app conserva solo un ano de historial y puede limpiar informacion antigua.

  • Los pagos de mas de 1 ano se eliminan.
  • Los prestamos de mas de 1 ano se eliminan.
  • Los clientes antiguos se eliminan solo si ya no tienen prestamos asociados.
  • La limpieza puede ejecutarse manualmente desde Configuracion.

9. Descarga de Excel

Hay un boton para descargar un archivo Excel con toda la informacion registrada.

  • Incluye hoja de resumen.
  • Incluye clientes.
  • Incluye prestamos.
  • Incluye pagos.
Uso sugerido Descarga el Excel antes de hacer limpiezas manuales, cambios grandes o eliminaciones.

11. Archivos y puntos tecnicos importantes

Si en el futuro alguien vuelve a tocar el proyecto, estos son los archivos clave para entenderlo rapido.

  • app/page.tsx: interfaz principal y flujo de la app.
  • app/api/export/excel/route.ts: exportacion de Excel.
  • app/api/maintenance/cleanup/route.ts: limpieza de historial.
  • app/api/auth/change-password/route.ts: cambio de clave.
  • lib/auth.ts: manejo de autenticacion.
  • SUPABASE_CONFIGURACION_APP.sql: capital inicial compartido.
  • SUPABASE_CREDENCIALES_APP.sql: tabla para credenciales editables.

12. Recomendaciones finales

  • Mantener la clave en un lugar seguro.
  • Descargar el Excel como respaldo con frecuencia.
  • Revisar que Vercel tenga las variables de entorno configuradas.
  • Probar la app en celular y computador despues de cada cambio importante.
